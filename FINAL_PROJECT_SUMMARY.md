## 📞 技术支持

### 常见问题

**Q: 页面导航不显示?**
A: 检查 CSS 文件是否正确加载，确保 `styles.css` 路径正确。

**Q: 如何添加新页面?**
A: 参考 `HEADER_TEMPLATE.html` 的结构，复制侧栏和 topbar 的 HTML 代码。

**Q: 如何修改导航菜单?**
A: 编辑各页面中的 `<nav class="nav">` 部分即可。

**Q: 响应式布局不工作?**
A: 确保在 `<head>` 中有 `<meta name="viewport">` 标签。
